- set `python.include_path` and `python.bin_path` properties in build_settings.yml file.

Your working dir is here (datalore-plot/python-package).


- build package:

$ (cd .. && ./gradlew :python-package-build:build)
or
$ python setup.py update_js bdist_wheel


- install `datalore-plot` package to currently active python environment:

$ pip uninstall datalore-plot
$ pip install --no-index --find-links=dist/ datalore-plot


To test plot in Jupyter in `connected` mode:
--------------------------------------------
Start local web-server to serve our js scripts:
    - cd to datalore-plot/js-package/build/dist
    - python -m http.server 8080


Sample bar plot:
----------------
from dlrplot import *
data = dict(time=['Lunch', 'Lunch', 'Dinner', 'Dinner', 'Dinner'])
p = ggplot(data) + geom_bar(aes(x='time', fill='..count..'))
p += scale_fill_hue()
p
